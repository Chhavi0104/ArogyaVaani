ArogyaVaani: AI-powered Vernacular Chatbot & QR-Based Health Records System

Overview:
ArogyaVaani is an AI-powered low-cost diagnostic system that provides symptom triage, medical guidance, and health records using a QR-based system. It supports multiple Indian languages (English, Hindi, etc.) and works on low-resource devices.

Features:
1. AI Chatbot for Symptom Checking & Guidance
2. Multilingual Support (English, Hindi, etc.)
3. Secure QR-Based Health Record Storage
4. Works on Low Infrastructure
